苹果mac os x下面软件的安装

Mac OS X下软件有四种安装方式。

  1. 最简单的拖拽安装，很多软件通过这种方式， 用户只需要把软件包拖动到本地程序目录就可以使用。
    它的原理主要是根据苹果系统能够自动动态链接到需要的类库， 并且程序的配置信息会在动在当前用户的Application Support文件夹创建。
  2. 通过安装程序安装，这种方式和windows下的安装包类似，一般文件结尾是.pkg， 通过向导提示用户一步一步的安装。一般会在系统的多个地方添加文件， 因此不能直接复制到一个位置。
  3. fink 是为了把unix下面的各种软件移植到Mac OS X下面而构建的， 它专门用于安装、管理各种软件包。一般安装位置在/sw 下。常用命令包括:
    1. fink configure 配置fink，包括下载目录、代理、下载站点、 是否打开unstable等，它会保存文件到/sw/etc/fink.conf
    2. fink apropos <package> 查找软件包
    3. fink list 列出所有的软件包
    4. fink install <package> 安装指定的软件包
  4. MacPorts 是和fink类似的另外一个软件包安装和管理工具

编译注意：

fink 提供了pkg-config命令，可以得到编译类库的各种选项，包括：

  1. pkg-config –list-all 列出当前已经安装的软件包/类库
  2. pkg-config –libs <libname> 列出编译引用选项
  3. pkg-config –cflags <libname> 列出编译其他选项，一般包括 -I

MacVim

设置透明

  1. 参数设置->勾选 Use experimental renderer.
  2. set Transparency=6
  3. Starting MacVim from the command line? 
    alias 'gvim'='/Applications/MacVim.app/Contents/MacOS/Vim -g'

设置为在新 tabs 中打开文件，在终端执行命令： 

defaults write org.vim.MacVim MMOpenFilesInTabs 1

对于 MacVim 7.3，全局菜单：MacVim -> Perferences -> General Open files from applications: in the current window, with a tab for each file.

环境变量

  1. 系统环境变量
    1. 终端 
      $ sudo vi /etc/profile
    2. 加入： 
      export PATH="$PATH:/path/you/need"
    3. 重启终端
  2. 用户环境变量
    1. 终端 
      $ sudo vi ~/.profile
      同上。
