Dopod 577W GPRS 设置

  1. 设置 -> 链接 -> GPRS
  2. 如果有的话，删除所有的
  3. 菜单 -> 添加
    1. 描述：CMNET(可随意)
    2. 链接到：Internet
    3. 连接点：CMNET
  4. 菜单 -> 添加
    1. 描述：CMWAP(可随意)
    2. 链接到：WAP网络
    3. 连接点：cmwap
  5. 另外可以下载GPRS设置包，安装即可。
